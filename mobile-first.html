Media Queries - Abordagem Mobile First

Conceitos

São regras especiais em que criamos alterações na página de acordo com determinadas condições.

Tabém são conhecidas como breakpoints (pontos de adaptação).

Aplicação

Media Queries são aplicadas de acordo com a estratégia usada para design responsivo.

  • Mobile First: telas pequenas para telas grandes
  • Desktop First: telas grandes para telas pequenas
